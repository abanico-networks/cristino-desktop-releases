## ¿Qué es Cristino Desktop?

Cristino Desktop es una aplicación nativa para macOS que permite optimizar imágenes de forma rápida y sencilla utilizando la [API Cristino](https://cristino.abanico.info), potenciada por ShortPixel.

La aplicación está desarrollada por **Abanico Networks** y es de uso interno para el equipo y los clientes de la agencia.

## Características

- Arrastra imágenes directamente a la ventana para optimizarlas
- Compresión Lossy, Glossy o Lossless
- Conversión a WebP y/o AVIF
- Eliminación de metadatos EXIF
- Sobrescribe el original o guarda junto a él con sufijo `_optimizada`
- Modo claro, oscuro y sistema
- Actualizaciones automáticas
- Firma y notarización Apple — compatible con Gatekeeper sin advertencias

## Requisitos

- macOS 12 Monterey o superior
- Intel o Apple Silicon (universal binary)
- Clave API de Cristino (solicitarla a Abanico Networks)

## Instalación

1. Descarga el archivo `.dmg` desde la sección [Releases](../../releases)
2. Abre el `.dmg` y arrastra **Cristino** a la carpeta Aplicaciones
3. Abre la app e introduce tu clave API en **Preferencias → API**
